# nvim

This package contains my Neovim configuration.

## To stow (dry run):

    stow -n -t "$HOME" nvim

## To stow (real):

    stow -t "$HOME" nvim

- Symlinks nvim config to $HOME/.config/nvim
- For Linux/macOS with XDG config directory

## Notes

- Requires Neovim 0.5+.
- Edit files in this directory to update your config.
